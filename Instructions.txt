Instrucciones para Ejecutar el Proyecto:

Sigue estos pasos para ejecutar el proyecto y permitir que los cambios se sincronicen correctamente.

Paso 1: Abre dos terminales

Terminal 1:

Ejecuta el siguiente comando para construir el proyecto:

npm run build

Ejecuta npx tsc para compilar los archivos TypeScript:

npx tsc

Finalmente, ejecuta el servidor con:

npm start

Esto iniciará el servidor en http://localhost:4000.

Terminal 2:

Ejecuta el siguiente comando para monitorear los archivos en dist/api/data/ y sincronizarlos automáticamente con los archivos originales:

npm run watch-json

Esto usará nodemon para observar los cambios en los archivos y ejecutará npm run sync-data automáticamente cada vez que se modifiquen los archivos en dist/api/data/.
